# 实验报告：基于CNN的MNIST手写数字识别实验

## 一、任务一：极简CNN学习与运行
### 1. 文章学习摘要
本次实验学习了微信公众号文章《计算机视觉》第10篇中关于**极简卷积神经网络（CNN）识别手写数字**的内容，核心要点如下：

- **CNN结构要点**：
  极简CNN以“卷积层-池化层-全连接层”为基础架构：
  - 卷积层：通过可学习的卷积核提取图像的边缘、纹理等低级特征，搭配ReLU激活函数引入非线性。
  - 池化层：通过下采样降低特征图尺寸，减少计算量并增强特征鲁棒性，常用2×2最大池化。
  - 全连接层：将二维特征图展平为一维向量，映射为10个数字类别的概率输出。

- **训练流程**：
  1. 数据准备：加载MNIST数据集，进行归一化处理并划分训练集与测试集。
  2. 模型定义：搭建包含卷积、池化、全连接层的极简CNN模型。
  3. 训练配置：选择交叉熵损失函数与Adam优化器，设置学习率、批量大小等超参数。
  4. 迭代训练：多轮迭代执行前向传播、损失计算、反向传播与参数更新。
  5. 模型评估：每轮训练后在测试集上计算分类准确率，验证模型泛化能力。

### 2. 实验配置与结果

| 层类型 | 输入尺寸 | 输出尺寸 | 关键参数 | 参数量 |
| :--- | :--- | :--- | :--- | :--- |
| 输入层 | 1×28×28 | 1×28×28 | - | 0 |
| Conv1 | 1×28×28 | 6×28×28 | 6个5×5卷积核，padding=2 | 156 |
| AvgPool1 | 6×28×28 | 6×14×14 | 2×2池化核 | 0 |
| Conv2 | 6×14×14 | 16×10×10 | 16个5×5卷积核 | 2416 |
| AvgPool2 | 16×10×10 | 16×5×5 | 2×2池化核 | 0 |

| Flatten | 120×1×1 | 120 | - | 0 |
| FC1 | 120 | 84 | - | 10164 |
| FC2 | 84 | 10 | - | 850 |

## 三、极简CNN与LeNet-5对比分析
在相同数据集与训练配置下，对两个模型的关键指标进行对比：

| 对比项 | 极简CNN | LeNet-5 |
| :--- | :--- | :--- |
| 测试集准确率 | 98.52% | 98.87% |
| 参数量 | ~160k | ~62k |
| 训练耗时 | 较长 | 较短 |

- **准确率对比**：LeNet-5的测试准确率略高于极简CNN，这得益于其经典的层级结构设计，对MNIST手写数字的特征提取更具针对性，池化与卷积层的搭配也更适配该任务的图像尺寸与特征分布。
- **参数量与效率对比**：LeNet-5的参数量仅为极简CNN的约1/2，模型更精简，训练速度更快，同时保持了更高的分类性能，体现了早期CNN设计的高效性与实用性。
